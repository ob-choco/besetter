### 5. Automatic Personal Information Collection Devices

BESETTER uses cookies and sessions to provide personalized services to users. A cookie is a small text file sent by a server operating a website to your browser and stored on your PC's hard disk.

A session refers to the storage of user information on the server during the user's connection time for service operation. When you visit a website, cookies and sessions help make your website access and use more convenient by allowing the website server to read the contents and maintain your settings.

Additionally, they enable the provision of optimized customized services by informing us of your website visit history, usage patterns, or areas of interest. Users have the choice regarding cookie installation and can set their web browser options to allow all cookies, to confirm each time cookies are stored, or to reject all cookie storage.

However, please understand in advance that there may be difficulties in using some services that require login if you reject cookie installation.

### 6. Disposal of Personal Information

Personal information is disposed of without delay when the collection and usage purposes for which consent was given are achieved, except when it needs to be retained according to laws or internal policies. When the collection and usage purposes for which consent was given are achieved, the user's personal information is moved to a separate database (or a separate file cabinet for paper records) and kept for a certain period as stipulated by relevant laws before being disposed of.

Personal information printed on paper is destroyed by shredding or incineration, and personal information stored in electronic file format is deleted using technical methods that prevent recovery of the records.

However, records of improper service use are retained for one year according to internal policies to prevent disputes arising from fraudulent service use, and then disposed of using the methods described above.
