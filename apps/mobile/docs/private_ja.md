# プライバシーポリシー

BESETTERプライバシーポリシー

BESETTERは、ユーザーの個人情報を管理・保護する重大な責任を持っています。約束します。皆様の大切な個人情報は、快適で良いサービスを提供するために必要な情報のみを利用いたします。BESETTERは、常に関連法令に従い、プライバシーポリシーを丁寧にご案内いたします。

### 1. プライバシーポリシーの意義

BESETTERは、情報通信網利用促進および情報保護などに関する法律、個人情報保護法、通信秘密保護法、電気通信事業法など情報通信サービス提供者が遵守すべき韓国の関係法令および個人情報保護規定、ガイドラインを遵守し、関連法令に基づくプライバシーポリシーを定め、ユーザーの権益保護に最善を尽くしています。本プライバシーポリシーは、BESETTERが提供するサービス（以下「サービス」という）に適用されます。

### 2. 収集する個人情報

BESETTERは、ユーザーがサービスに登録する際、またはサービス利用過程でウェブサイトや個別アプリケーション、プログラムなどを通じて、以下のようなサービス提供に必要な最小限の個人情報を収集しています。個別サービスの利用過程で追加的に個人情報の収集が発生する場合は、収集する個人情報について別途ご案内し、これに基づきユーザーの同意を得てサービスを提供します。

[会員登録時] メール登録時（携帯電話番号、メールアドレス、パスワード、ニックネーム、生年月日、性別）、SNS登録時（携帯電話番号、ユーザー固有識別子、メールアドレス、ニックネーム、生年月日、性別）、端末情報

その他、サービス利用に応じてスマートフォンなど端末のアドレス帳に保存された連絡先情報（第三者の電話番号、名前）、クッキー、サービス利用記録（訪問日時、IP、ログイン時間、ユーザーロール、国、都市など）、端末情報（OS、画面サイズ、デバイスID、端末機種、端末モデル名など）、CPUおよびLANカード情報などの運営システムおよびハードウェア環境情報などの情報が自動的に生成され収集される場合があります。

### 3. 収集した個人情報の利用

会員管理、サービス開発・提供および向上、安全なインターネット利用環境構築など、以下の目的でのみ個人情報を利用します。コンテンツなど既存のサービス提供（広告含む）に加え、人口統計学的分析、サービス訪問および利用記録の分析、個人情報および関心に基づくユーザー間関係の形成、知人および関心事などに基づくカスタマイズされたサービス提供など、新規サービス要素の発掘および既存サービス改善のために個人情報を利用します。

法令および利用規約に違反するメンバーに対する利用制限措置、不正利用行為を含むサービスの円滑な運営を妨げる行為に対する防止および制裁、アカウント盗用および不正取引防止、規約改正などの告知事項伝達、紛争調整のための記録保存、苦情処理などユーザー保護およびサービス運営のために個人情報を利用します。

有料サービス提供に伴う本人認証、購入および料金決済、商品およびサービスの配送のために個人情報を利用します。イベント情報および参加機会提供、広告情報提供などマーケティングおよびプロモーション目的で個人情報を利用します。

サービス利用記録と接続頻度分析、サービス利用に関する統計、サービス分析および統計に基づくカスタマイズされたサービス提供および広告掲載などに個人情報を利用します。セキュリティ、プライバシー、安全面でユーザーが安心して利用できるサービス利用環境構築のために個人情報を利用します。

### 4. 個人情報の提供および委託

BESETTERは、ユーザーの別途同意がある場合や法令に規定された場合を除き、個人情報を第三者に提供しません。今後サービス利用過程などで別途同意する場合や法令に規定された場合を除き、ユーザーの個人情報を目的範囲を超えて利用したり、第三者に提供または共有することはありません。

ただし、第三者のサービスが連携して提供される場合、利用に必要な範囲内で皆様の同意を得た後、以下のようにサービス運営と開発や顧客相談の目的で個人情報を第三者に提供することがあります。

### 5. 個人情報自動収集装置

BESETTERは、ユーザーにパーソナライズされたサービスを提供するためにクッキー（cookie）とセッション（session）を使用します。クッキーとは、ウェブサイトを運営するサーバーがユーザーのブラウザに送信する小さなテキストファイルで、PCのハードディスクに保存されます。

セッションとは、サービス運営に利用されるサーバーがユーザー接続時間中にユーザー情報をサーバーに保存することを意味します。クッキーとセッションは、ユーザーがウェブサイトを訪問する際、ウェブサイトサーバーが内容を読み取り環境設定を維持できるようにすることで、ユーザーのウェブサイトアクセスと便利な使用を助けます。

また、ユーザーのウェブサイト訪問履歴、利用形態や関心分野を知らせることで、これを通じた最適化されたカスタマイズサービスの提供を可能にします。ユーザーはクッキー設定に関する選択権を持っており、ウェブブラウザのオプションを設定することにより、すべてのクッキーを許可したり、クッキーが保存されるたびに確認を経たり、あるいはすべてのクッキーの保存を拒否することもできます。

ただし、クッキー設定を拒否した場合、ログインが必要な一部のサービスの利用には困難がある場合がありますことをあらかじめご了承ください。

### 6. 個人情報の破棄

個人情報は、同意を得た個人情報の収集および利用目的が達成された場合、法令または内部方針により保存する必要がある場合を除き、遅滞なく破棄されます。ユーザーの個人情報は、同意を得た収集および利用目的が達成された場合、別途のDBに移動され（紙の場合は別途の書類箱）、関係法令で定めた一定期間保管された後、破棄されます。

紙に出力された個人情報は、シュレッダーで裁断するか焼却により破棄し、電子的ファイル形式で保存された個人情報は、記録を再生できない技術的方法を使用して削除します。

ただし、サービスの不正な利用による紛争を防止するための内部方針に従い、サービスの不正利用記録は1年間保存した後、上記の方法で破棄されます。

### 7. ユーザーおよび法定代理人の権利と行使方法

ユーザー（14歳未満の場合は保護者）は、いつでも個人情報を照会したり修正することができ、収集・利用に対する同意撤回または会員退会をリクエストすることもできます。より具体的には、サービス内の設定機能による変更を、会員退会（同意撤回）のためにはサービス内の「会員退会」をクリックすれば済み、BESETTER顧客センター（contactus@olivebagel.com）に連絡すれば遅滞なく対応いたします。ユーザーのリクエストにより解約または削除された個人情報は、関連法と定められた規定に従って安全に処理し、それ以外の用途で閲覧または利用できないように処理しています。
