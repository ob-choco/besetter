API 설계

GET /places/nearby
Query:
  latitude:  number
  longitude: number
  radius:    number (default: 100m)

Response: 200
[{
  id, name, type,
  latitude, longitude,
  distance, createdBy
}]

// type=gym → 모두 반환
// type=private-gym → 본인만
POST /places
Body:
  name:      string
  latitude:  number
  longitude: number
  type:      "gym" | "private-gym"

Response: 201
{ id, name, type,
  latitude, longitude,
  createdBy }

Place 데이터 모델

collection: places
{
  _id:        ObjectId
  name:       String          // "더클라임 강남점"
  type:       String          // "gym" | "private-gym"
  latitude:   Number          // 37.516523
  longitude:  Number          // 127.019823
  createdBy:  ObjectId (ref)  // 최초 등록 userId
  createdAt:  Date
}

// index: 2dsphere on { latitude, longitude }
