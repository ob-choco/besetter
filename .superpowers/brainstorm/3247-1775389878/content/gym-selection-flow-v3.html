지도 구현 스택

패키지: google_maps_flutter (iOS + Android 동일)
무료 한도: 월 28,000 Dynamic Maps 로드

지도 동작:
  1. EXIF GPS 추출 → 초기 카메라 위치 + 핀 배치
  2. 유저가 핀 드래그 → onCameraMove로 좌표 업데이트
  3. "등록하기" 탭 → 최종 핀 위치의 latitude/longitude 전송

EXIF 없는 경우:
  - 기기 현재 위치로 fallback (geolocator 패키지)
  - 위치 권한 없으면 → 서울 시청 기본값 + 안내 문구

API 설계

GET /places/nearby
Query:
  latitude:  number (required)
  longitude: number (required)
  radius:    number (default: 100m)

Response: 200
[{
  id, name, type,
  latitude, longitude,
  distance, createdBy
}]

// type=gym → 모두에게 반환
// type=private-gym → 본인만
POST /places
Body:
  name:      string (required)
  latitude:  number (required)
  longitude: number (required)
  type:      "gym" | "private-gym"

Response: 201
{
  id, name, type,
  latitude, longitude,
  createdBy
}
