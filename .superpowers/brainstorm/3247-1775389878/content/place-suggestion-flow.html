수정 플로우 요약

[선택된 암장에서 "정보 수정 제안" 탭]
    ↓
┌─ type=gym
│   → 수정 제안 폼 (현재값 → 새값 비교)
│   → POST /place-suggestions
│   → "제안 접수됨" 안내
│   → 운영자: readAt (알림 열람) → status 변경 → reviewedAt
│
└─ type=private-gym (본인만)
     → 즉시 편집 폼
     → PUT /places/:id
     → 즉시 반영
