데이터 플로우

[사진 업로드] → EXIF GPS 추출 (lat, lng)
    ↓
[서버] GET /gyms/nearby?lat=37.51&lng=127.01&radius=100
    ↓
┌─ 결과 있음 → 리스트 표시 → 유저 선택 → gymId 연결
└─ 결과 없음 → "새 암장 등록" → POST /gyms { name, lat, lng } → gymId 연결

API 설계 (안)

GET /gyms/nearby
Query:
  lat: number (required)
  lng: number (required)
  radius: number (default: 100m)

Response: 200
  [{ id, name, lat, lng, distance }]
POST /gyms
Body:
  name: string (required)
  lat: number (required)
  lng: number (required)

Response: 201
  { id, name, lat, lng }
